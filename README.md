# dotfiles

## install

```sh
curl -L https://raw.githubusercontent.com/ykawa/dotfiles/develop/setup.sh | bash -
```

## (optional) restore cinnamon settings

```sh
dconf load /org/cinnamon/ < cinnamon_settings
```
